# 📻 Ultimate Hip-Hop Hub

Una aplicación web minimalista y ligera diseñada para melómanos del género urbano. Este reproductor conecta directamente con las APIs de **Archive.org** para ofrecer streaming aleatorio de miles de joyas del Rap, R&B, Instrumentales y Dancehall.

## 🚀 Características

* **6 Canales Temáticos:** Desde Rap clásico hasta Dancehall y Beats.
* **Streaming Inteligente:** Selección aleatoria de pistas basada en colecciones curadas.
* **Diseño Responsive:** Optimizado para dispositivos móviles y escritorio.
* **Modo Web-App:** Instalable en la pantalla de inicio de tu smartphone.
* **Zero Backend:** Funciona 100% en el navegador (HTML/CSS/JS).

| Canal | Contenido |

## 📲 Cómo usar en el móvil (PWA Style)

Para llevar tu radio siempre contigo como si fuera una App nativa:

1.  Abre la URL de tu **GitHub Pages** en Chrome (Android) o Safari (iOS).
2.  Pulsa el botón de **Compartir** o los **tres puntos** de configuración.
3.  Selecciona la opción **"Añadir a pantalla de inicio"**.
4.  ¡Listo! Aparecerá un icono en tu menú para acceder con un toque.

## 🛠️ Tecnologías utilizadas

* **HTML5 / CSS3:** Layout moderno con Grid y Flexbox.
* **JavaScript (Vanilla):** Lógica de conexión con la API de Metadata de Archive.org.
* **FontAwesome:** Iconografía técnica.
* **Archive.org API:** El motor que proporciona el contenido multimedia.
